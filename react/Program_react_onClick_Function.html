[React] function vs function() in react

I came across a problem in AddWordButton file, I tried to dispatch the addWord action to add a word in the redux. But I found that before I clicked the button, this function has already been run! The code is like this: 
            
    return <button onClick={dispatch(addWord(word, meaning))}>+</button>;
After I searched, I found that searching() is calling the function referenced by searching. Searching itself is just a reference to a function, it doesn't call the function.

So in reactjs, if I take onclick as an example:
  • if that is a function without parameter, you should the function like this.
    <button onClick="{searching}">Search</button>
  • if that is a function with parameter, you should the function like this.
    <button onClick={() => dispatch(addWord(word, meaning))}>+</button>
