When the child element's position is set to absolute, the parent element should set to relative. Or the layout will look like this:

Parent 1 : default position

Child 1 : parent is default position.

When the parent element is set to relative, it looks like this:

Parent 2 : position: relative;

Child 2 : parent position is relative.

When the parent element didn't set but the grandparent position is set to relative, it looks like this:

Grandparent 3 : position: relative;

Parent 3 : default position

Child 3 : grandparent position is relative.
