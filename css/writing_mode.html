Actually the direction in writing mode rl which indicates right to left and lr which indicates left to right doesn't show us what we expect. They are all block direction instead of character direction which means all the character are rotated clockwise. So to make it what we think, we need to rotate the text 180 degree.
We can see the inline direction goes from top to bottom to bottom to top now.
