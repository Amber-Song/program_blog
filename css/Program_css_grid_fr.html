[CSS] Using fr instead of percentage in grid

fr is a quite recent feature add in grid. Before this one came out, I sometimes use grid-template-columns: repeat(2, 50%); or grid-template-columns: 50px 50% 30% 20%; to devide the columns, but it didn't come out as I thought when I have grid-gap as part of the style. Because for repeat(2, 50%) when the percentage is used, it works in a way that 100% width is divided into two columns each of which is 50%, so when grid-gap is added, the actual width is more than 100%. You can tell from the result of grid-template-columns: 50px 50% 30% 20%; . But when fr is used, the width excluding the gap will be divided based on the ratio.
