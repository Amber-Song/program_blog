[CSS] id selector in external css file doesn't work! [Question mark]

I try to select the element and then apply the style on it in the external CSS file. But when I selected it by its id, it doesn't work. While I selected it by its class, it works.
When I wrote this blog's demo, I found that if you just want to select elements and change their style, both selecting by id and class works. If you want to apply the style to the elements when your mouse hover themselves, both selecting by id and class works. But if you want to apply the style to the elements when your mouse hover other elements, both of them doesn't work at all this time. If it doesn't work in the external CSS file, it doesn't work in the internal CSS file as well. There is no error. I still don't know why.

I searched online and found some possibilities for why it doesn't work.
  1. If you add space before :hover, that element will not change the style, but all of its descendant elements will change the style.
  2. If you want to change the style of other elements when you hover one element, the elements you can change is descendant elements and only siblings following it. If you don't add + before sibling element, it still doesn't work.
  3. For the tag a, a:hover isn't placed after a:link and a:visited.
And I think I found the reason that I didn't work for the above demo. When hovering the second element, the last element's style doesn't change, because only the sibling element directly following it will change. when the third element hovered, the last one changes, because it is directly following it.
