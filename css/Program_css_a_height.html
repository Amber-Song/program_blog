[CSS] How to set height and width for tag a!

I tried to set the height and width for the tag a. But it didn't work, the border still tightly wrapped the content. After I searched online, I found that if an element displayed as an inline element, any height and width properties would have no effect. So I set the display to block in order to make the setting of height and width work. Here is the demo:

After I tried, I found that not only the height and width of inline elements didn't work, Margin of that didn't work as well. And set the display to block or inline-block do help.
