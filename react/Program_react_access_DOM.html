[React] access DOM in reactjs

When I just switched from vanilla js to reactjs, I felt very it very different when I wanted to do something which needs access to DOM.
Like when I want to change the class of one element to another class, I used to write: document.getElementById('bid').className = 'show-more';
But in reactjs I can't get the node by getElementById function or anything like that. In reactjs, it need to use another way to achieve this by making the class or inner HTML or anything I want to change the state. Instead of picking the node and change it, I make it variable inside state and when I want to change it, I change the value of the variable.
