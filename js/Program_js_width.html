[JS] get width and height in JS

There are several properties that you can get from js.
  • window.innerwidth and innerheight return the content area of window.
  • window.outerwidth and outerheight return the whole browser window.
  • And for IE8 and earier, user clientwidth and clientheight as innerwidth and innerheight.
  • Using style to get width and height is to get the set width and height. The value for span tag will be none.
  • Element.clientWidth and Element.clientHeight include padding but exclude border, margin and scrollbars. It is zero for the inline element. It is int.
  • HTMLElement.offsetWidth and HTMLElement.offsetHeight include padding, border and scrollbars, exclude margin. It is zero for the hidden element.
  • Element.scrollWidth and Element.scrollHeight include padding but exclude border, margin and scrollbars. It is the content height.
