As it is said online that if the height is set by percentage, the value of the height depends on the height of the parent element. However, the body and html elements, in this case, have no default value and their heights depend on the total height of all the descendants' height which means height:100% in the div will not take the whole screen.
To solve this, you could also set the html and body to 100% or you could set the height to 100vh. vh relative to 1% of the height of the viewport.
